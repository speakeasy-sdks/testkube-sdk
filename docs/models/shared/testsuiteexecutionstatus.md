# TestSuiteExecutionStatus

optional status filter containing multiple values separated by comma


## Values

| Name       | Value      |
| ---------- | ---------- |
| `Queued`   | queued     |
| `Running`  | running    |
| `Passed`   | passed     |
| `Failed`   | failed     |
| `Aborting` | aborting   |
| `Aborted`  | aborted    |
| `Timeout`  | timeout    |
